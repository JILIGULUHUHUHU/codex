# Python 小游戏合集

这是一个使用 **Python + Tkinter** 编写的小游戏项目，可以在同一个窗口中切换不同游戏（贪吃蛇、俄罗斯方块）。

## 功能

- 游戏选择：贪吃蛇 / 俄罗斯方块
- 统一分数与状态显示
- 一键开始或重开
- 键盘即时操作

## 运行环境

- Python 3.10+
- Tkinter（标准库，通常随 Python 一起安装）

## 启动方式

```bash
python main.py
```

## 操作说明

### 贪吃蛇

- 方向键：控制移动

### 俄罗斯方块

- `←` / `→`：左右移动
- `↓`：加速下落
- `↑`：旋转方块
